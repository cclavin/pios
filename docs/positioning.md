# PIOS Positioning

PIOS is an AI Project Execution Contract.

It defines deterministic phase gates and artifact checks so AI-assisted projects can finish with verifiable state transitions.

## What PIOS Is

- A tool-agnostic execution contract for software repositories
- A CLI-backed gatekeeper for phase transitions (`pios validate`)
- A set of artifact templates for project state, plan, and tasks

## What PIOS Is Not

- Not an agent runtime platform
- Not a model orchestration layer
- Not a memory backend, vector store, or hosted control plane

## PIOS vs Runtime Infrastructure

Runtime platforms execute model/tool loops.
PIOS defines completion criteria and validates whether repository artifacts satisfy those criteria.

Use runtime infrastructure to do the work.
Use PIOS to prove the work is done.

## Core Outcomes

PIOS optimizes for:

- Finishability
- Portability
- Measurability
- Trust
