# Backtesting PIOS

Goal: Validate PIOS reduces back-and-forth and increases finish rate.

## Compare
Run 2–3 small projects with:
- Baseline (your normal workflow)
- PIOS workflow

Optionally add comparators:
- Spec-driven workflow you already use
- Another framework/tool

## Metrics to Track
- time_to_scaffold_minutes
- clarification_turns
- rework_events
- context_resets
- finish_rate (0/1)
- quality_notes (short)

Use `metrics.template.json` per run.
